# LangChain 介绍

LangChain 是一个基于 LLM 构建智能体和应用程序的框架。

只需要极短的代码就可以连接到 OpenAI、DeepSeek 等平台的 LLM 模型。

提供快速连接LLM和构建智能体的方法。

## 相关生态

- LangChain
- LangGraph
- Deep Agents

## 环境

- Python 3.10 或以上版本
- pip 包管理器

**安装常用包**

```bash
# 安装 LangChain 核心库
pip install -U langchain
# 安装 OpenAI 集成
pip install -U langchain-openai
# 安装 Anthropic 集成
pip install -U langchain-anthropic
```
