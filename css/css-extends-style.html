可继承样式 ​

CSS 中有些样式是可继承的。

但这些样式并非在任何时候都会被继承，一些标签无法继承这些可继承样式。

可继承的属性：

  • color 文字颜色
  • font-size 字体大小
  • font-family 字体
  • line-height 行高
  • ul / li / dl / dd / dt 项目列表

不继承 color 的标签 ​

  • a 超链接
  • button 按钮
  • input 输入框
  • select 下拉框
  • textarea 文本域

不继承 font-size 的标签 ​

  • button 按钮
  • input 输入框
  • select 下拉框
  • textarea 文本域
