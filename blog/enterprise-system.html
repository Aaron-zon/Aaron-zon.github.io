常见的企业信息化系统（OA/ERP/MES/WMS） ​

系统核心对象用户数据粒度核心作用
ERP财务、订单、库存、成本管理层粗（批次/订单）企业决策、业务管理
MES工单、工序、工人、设备车间细、实时（工序、工时）生产执行
WMS库存、库位、物料仓库细、实时（单件/批次/库位）仓储物流管理
OA审批、流程、人事全员流程记录内部协同办公

OA 系统 ​

管理 人、流程、审批这类办公行为的系统

OA = Office Automation(办公自动化系统)

核心特点：

  • 面向人：员工、管理者
  • 面向流程：请假、报销、采购审批等
  • 非核心业务：不直接管财务、产生结果、但支撑企业高效运作

简单来说： OA系统是企业内部用来管理办公事务、审批流程、人事考勤、文档协作的系统,他的优势是高效、标准化、可追溯。

OA系统的经典模块 ​

模块功能示例
流程管理 / 审批请假、报销、合同审批、采购审批
公文管理公文收发、文档归档、通知公告
考勤 / 人事打卡、请假、加班记录、员工信息管理
协同办公任务分配、会议管理、日程、即时通讯
知识管理企业内部知识库、经验分享、文档模板

OA系统的核心功能 ​

1.流程标准化

不用纸质审批，自动流转到负责人。

2.提高效率

员工提交 → 系统提醒 → 审批完成 → 自动归档

3.记录可追溯

谁审批、什么时候审批、审批结果，一目了然

4.辅助决策

通过流程数据和报表分析办公效率

举个例子 ​

请假流程

1.员工在 OA系统里提交请假申请

2.系统自动通知领导审批

3.审批通过 → 自动记录在考勤系统

4.系统生成报表供 HR 查看

ERP 系统 ​

管理 钱、货、生产、业务结果 的系统

ERP = Enterprise Resource Planning(企业资源计划系统)

ERP 就是用一套系统，把企业的人、财、物、产、供、销等核心业务全部“管在一起”。

它的核心目标是：

提高管理效率、减少重复工作、让数据统一、实时、可追溯。

ERP 常见模块 ​

1.财务管理

  • 总账
  • 应收/应付
  • 成本核算
  • 固定资产
  • 财务报表

2.采购管理

  • 采购申请
  • 采购订单
  • 供应商管理
  • 到货、入库、队长

3.销售管理

  • 客户管理
  • 销售订单
  • 发货
  • 开票
  • 回款

4.库存/仓储管理

  • 入库/出库
  • 库存盘点
  • 批次/序列号管理
  • 库存预警

5.生产管理（制造型企业）

  • 生产计划
  • 物料需求计划（MRP）
  • 工单管理
  • BOM（物料清单）

ERP 核心价值 ​

1.提高管理效率

  • 自动流转业务流程（如采购 → 入库 → 财务核算）
  • 避免重复录入和人工对账
  • 部门协作更顺畅

2.数据统一和准确

  • 所有业务数据在同一个系统中
  • 避免各部门信息孤岛
  • 实时反映库存、财务、销售等情况

3.降低成本

  • 精准掌握库存，减少积压
  • 生产计划和采购计划更合理
  • 人力成本减少（自动化处理报表、凭证）

4.支持决策

  • 可生成财务、库存、销售、生产报表
  • 通过分析数据预测需求、优化生产和采购
  • 高层可以快速了解企业运营状况

5.可追溯与合规

  • 每笔业务操作有记录
  • 容易追踪库存、财务、生产等异常
  • 满足审计和监管需求

举个例子 ​

场景：生产型企业，生产 1000 件产品

1.销售下单

  • 客户下订单 1000 件产品
  • ERP 生成销售订单，通知财务和生产计划

2.物料需求计算

  • ERP 根据 BOM（物料清单）计算需要多少原料
  • 判断库存不足的原料，生成采购计划

3.采购和入库

  • ERP 下采购订单
  • 物料到货 → 入库登记 → 库存自动更新

4.生产计划下发

  • ERP 下发生产工单（可对接 MES）
  • MES 在车间执行生产

5.成本核算

  • ERP 根据实际消耗的物料、人工、机器工时，自动计算生产成本
  • 成本入账、利润自动生成

6.销售发货

  • 成品库存更新
  • ERP 生成出库单和发票
  • 财务自动生成应收账款

成熟的ERP系统 ​

国际的

  • SAP
  • Oracle ERP
  • Microsoft Dynamics

国内的

  • 用友
  • 金蝶
  • 浪潮

ERP适合什么企业？ ​

  • 中大型企业
  • 业务流程复杂
  • 多部门协同
  • 对财务、库存、生产要求高

MES 系统 ​

MES = Manufacturing Execution System（制造执行系统）

MES 是连接 ERP 和车间生产的‘桥梁’，负责把生产计划落实到每一道工序，并实时监控生产过程。

MES 经典模块 ​

1.生产计划下发

将 ERP 下发的生产订单，拆分为工单、工序、任务

2.工单执行

记录工序开始/结束时间，工人操作记录

3.设备管理

设备状态监控、故障记录、产能统计

4.质量管理

过程检、首检、终检、不良品追踪

5.物料管理

生产物料领用、半成品/成品流转

6.数据采集

采集工时、产量、设备数据、环境参数

报表分析

生产效率、良品率、工时成本分析

MES 核心价值 ​

  • 实时管控生产：车间的每个工序都可以被监控、记录
  • 提高生产效率：发现瓶颈，优化产线
  • 质量可追溯：不良品来源、批次、工序一目了然
  • 数据闭环：生产数据回流 ERP，支持成本核算和决策

举个例子 ​

假设生产1000件产品

1.ERP：生成生产订单，计算所需物料

2.MES：

  • 下发工单到车间
  • 工人按工序操作，刷卡报工
  • 设备实时记录产量
  • 不良品记录、反工

3.WMS：生产完成后，成品入库，记录库位、批次

4.ERP：核算实际成本、库存变化，生成财务报表

WMS 系统 ​

WMS = Warehouse Management System（仓库管理系统）

WMS 专注于仓库物料的管理，实现入库、存储、出库、盘点、调拨的数字化和可追溯。

WMS 经典模块 ​

1.库存管理

采购入库、生产入库、批次/序列号管理、验收入库

2.库存管理

实时库存查询、枯萎管理（货架/区域）、库存调拨、库存预警

3.出库管理

拣货、配货、打包、发货单管理、出库扫描校验

4.盘点管理

定期盘点、抽盘、盘点差异处理、库存异常报警

5.物流对接（可选）

与快递、运输、自动化立库/WCS对接

6.保质期管理

先进先出（FIFO）、批次、有效期管理

WMS 核心价值 ​

1.库存准确：实时掌握仓库每件物料的位置和数量

2.仓库高效：减少人工找货、错发、漏发

3.可追溯：每批物料来源、去向清晰

4.辅助生产/销售：保证 MES / ERP 的物料供应和成品发货

举个例子 ​

生产 1000 件产品：

ERP：生成生产订单，计算库存

MES：下发工单到车间，生产 1000 件

WMS：

  • 成品入库，记录批次、库位
  • 库存实时更新
  • 拣货出库，生成发货单

ERP：核算实际库存、成本、财务入账

💡 MES 负责生产过程，WMS 负责物料流转，ERP 负责管理和核算
